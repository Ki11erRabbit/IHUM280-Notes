Wednesday 10/20/2021

Basic Models for Monetization

  • Purchasing the game
  • Subscription Model
  • Free-to-Play (in app purchases)

Purchasing the Game

  • using money from individual sales
  • on PC you have to deal with piracy
  • in Sid Meier's Pirates! it asks you a question that required possession or knowledge of the game manual with 12 answers. The game doesn't tell you if you got the question wrong. If you got it wrong, the game would maroon you midway through the game

Rise of Mobile Games

  • selling really cheap games
  • $200mil Angry Birds

Downloadable Content (DLC)

  • purchasing additonal content
  • makes purchases more spread out

Subscription Model

somewhat like arcades although I think that arcades are more like in app-purchases

MMOs

Fortnite

Xbox Game Pass

  • game service that allows you to play games you don't own on your hardware
  • titles can come and go
  • service plans on going streaming

Free-to-Play Model

financed by in-app purchaes

credit card number becomes the cheat code

usually used in simple and casual games

Examples:

  • Candy Crush
  • Clash of Clans ($2.4 million a day)

Uncertain Purchases

  • loot boxes, gacha games
  • baseball cards and other trading card games (randomized packs)
  • Whale: slang term for people who spend a ton of money on in-app purchases (also common in gambling)

Examples:

  • Middle-Earth Shadow of War
  • Battlefront II: Got pressure from fans from a boycott and the loot boxes got removed and Disney revoked the exclusive license for Starwars games
  • Genshin Impact: Can take some time before needing to use loot boxes. Makes it so that every few rolls you are guaranteed something good
  • FIFA: uses a baseball card system. Assemble the best team from paying for lootboxes, Belgium banned minors from the game
  • Overwatch: loot box only is for cosmetics

From Ramin Shokrazade:

  • Intermediate Currencies: gold bars in Candy Crush
  • "Fun Pain" and Relief: so close to winning but if you pay you can continue
  • Skill Game vs Money Game: game that requires skill vs game that requires money
  • Reward Removal: you get a reward every level but if you die before getting to the end you have to start over and lose all of the rewards
  • Ante Game: put in more money to do well
  • Gating: blocking progress with money
